- Hook: AI tools are moving from chat to doing tasks.
- Define MCP in one sentence (standard way for AI apps to connect to tools/data).
- Current event angle: MCP is now under the Linux Foundation’s Agentic AI Foundation.

- AI app (client/host) connects to an MCP server (connector).

- Why it matters: less custom glue code, easier reuse of integrations.

Speaker 3 (1:55–2:50) — Practical impact + basic guardrails
- Remote servers: teams can share integrations across machines.
- Guardrails: trusted servers only, limit permissions, logs/review for actions.
